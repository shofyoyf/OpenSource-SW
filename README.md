**top
의미: 리눅스 시스템의 CPU, 메모리 사용량 등 전반적인 상황을 실시간으로 모니터링할 수 있는 명령어


Tasks: 현재 실행 중인 프로세스 수
CPU(s): CPU 사용률
Mem: 물리적 메모리 사용량
Swap: 스왑 메모리 사용량

**ps
의미: 현재 실행 중인 프로세스와 상태를 출력하는 명령어

<옵션>
ps -e: 현재 사용자뿐만 아니라 다른 사용자들이 구동시킨 모든 프로세스를 보여줌
ps -f: 프로세스 간 상속관계를 트리형식으로 보여줌
ps -l: -f보다 더 자세한 정보 보여줌

**jobs
의미: 현재 쉘 세션에서 실행 중이거나 일시 중지된 작업을 확인할 때 사용하는 명령어

<옵션>
jobs -l: 프로세스 그룹 ID를 state 필드 앞에 출력
jobs -n: 프로세스 그룹 중에 대표 프로세스 ID를 출력
jobs -p: 각 프로세스 ID에 대해 한 행씩 출력
jobs -r: 실행중인 JOB만 출력
jobs -s: 중지된 JOB만 출력

<상태값>
Running: 작업이 계속 진행 중
Done: 작업이 완료되어 0으로 반환하고 종료
Stopped: 작업이 일시 중단됨


**kill
의미: 특정 프로세스나 프로세스 그룹에 시그널을 보내 프로세스를 종료하는 데 사용하는 명령어

<옵션>
kill -l: 사용 가능한 모든 신호 목록을 숫자로 표시함
kill -9 (SIGKILL): 프로세스를 강제로 종료시킴
kill -15 (SIGTERM): 프로세스에 종료를 요청하는 신호를 보냄, 프로세스가 정리할 시간을 줌
kill -2 (SIGINT): 프로세스의 작업을 즉시 종료시키려는 신호
